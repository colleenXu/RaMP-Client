Brief Description

RaMP is an integrative pathway database harmonizing analyte (genes, proteins, and metabolites) and pathway information from the following data sources: {{version.dataSourceName}} {{version.dataSourceVersion}} , .
Entity {{row.entity}} {{column}} {{row.counts[column] | number}}
This application allows users to interface with RaMP without any programming experience. The functionalities currently supported are:
  • Perform a pathway enrichment analysis given a list of analyte names or IDs,
  • Retrieve the analytes involved in a given list of pathways,
  • Retrieve the pathways containing a given list of analytes,
  • Find the analytes involved in the same reaction, and
  • Find the ontologies for a given list of metabolites or viceversa.
Of note, RaMP does not import all the information contained in these databases. Instead, RaMP includes pathway names and the genes and metabolites they include. RaMP is a work in progress and we will be expanding its content (reaction-level information, non-human compounds and genes, etc.).
More details can be found in our manuscript, and an example usage can be found in this chapter.
