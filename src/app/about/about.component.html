RaMP: Relational Database of Metabolomics Pathways

To use this software on your local machine, you must have the RaMP SQL database installed locally. For instructions on how to do so, please refer to the RaMP-DB GitHub repository.
RaMP currently integrates information from 4 different databases:
  • Kegg
  • Reactome
  • HMDB
  • wikipathways
Of note, RaMP does not import all the information contained in these databases. Instead, RaMP includes pathway names and the genes and metabolites they include. RaMP is a work in progress and we will be expanding its content (reaction-level information, non-human compounds and genes, etc.).
More details can be found in our manuscript
For questions or issues, please submit an issue on our GitHub site at https://github.com/ncats/RaMP-DB
